# DevQuiz — Developer Interview Prep Quiz App 🧠

An interactive quiz application to help developers prepare for technical interviews. Covers **Python**, **JavaScript**, **SQL**, and **React JS** — the exact stack required for modern full-stack roles.

🔗 **Live Demo:** [https://sravankumarmekala.github.io/devquiz/](https://sravankumarmekala.github.io/devquiz/)

---

## 📸 Preview

> A clean, editorial-style quiz app with code snippets, instant feedback, explanations, and streak tracking.

---

## ✨ Features

- 🐍 **Python** questions — lambda, list references, exception handling
- ⚡ **JavaScript** questions — typeof, coercion, closures, JSON
- 🗄️ **SQL** questions — HAVING, DISTINCT, JOINs, aggregations
- ⚛️ **React JS** questions — useState, props, component lifecycle
- 🎯 **Category filter** — Study one topic at a time or all together
- 💡 **Instant explanations** — Every answer has a detailed "why"
- 🔥 **Streak counter** — Tracks consecutive correct answers
- 📊 **Score tracking** — Live score displayed throughout the quiz
- 🔀 **Shuffled questions** — Different order every time you play
- 🏆 **Result screen** — Performance summary with emoji feedback
- 📱 **Fully responsive** — Works on all screen sizes

## 🧠 Concepts Demonstrated

- **JavaScript State Management** — Pure JS quiz engine without any framework
- **Array Manipulation** — `filter`, `map`, `shuffle` (Fisher-Yates algorithm)
- **Event-Driven Architecture** — Click handlers, dynamic UI updates
- **Conditional Rendering** — Show/hide elements based on quiz state
- **CSS Custom Properties** — Consistent theming with `var(--color)`
- **Responsive Grid Layout** — CSS Grid with media query breakpoints
- **Code Display** — Syntax-highlighted code snippets using CSS spans
- **UX Design** — Feedback states (correct/wrong), progress, streaks

## 📚 Topics Covered

### Python
- Mutable vs Immutable types, list references
- `try / except` exception handling
- Lambda functions, list comprehensions
- Built-in functions — `len()`, `range()`, `type()`

### JavaScript
- Type coercion and `typeof` quirks
- `===` vs `==` strict vs loose equality
- JSON methods — `parse()` and `stringify()`
- Async basics — `fetch`, Promises, async/await concepts

### SQL
- `WHERE` vs `HAVING` clause differences
- `SELECT DISTINCT` for unique values
- Aggregate functions — `SUM`, `COUNT`, `AVG`
- `GROUP BY`, `ORDER BY`, `JOIN` basics

### React JS
- `useState` and `useEffect` hooks
- Props — parent to child data flow
- Component-based architecture
- Conditional rendering in JSX
